# Promtia

A lightweight web chat widget that uses OpenAI’s **Responses API** to behave like your custom assistant. The server keeps your API key safe; the browser only talks to your server.

## Requirements
- Node.js 18+ (20+ recommended)
- An OpenAI API key

## Quick start
1. Copy `.env.sample` to `.env` and fill in your key.
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
4. Open `http://localhost:3000`
